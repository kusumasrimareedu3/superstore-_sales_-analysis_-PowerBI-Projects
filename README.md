# 📊 Superstore Sales Analysis Using Python, SQL and Power BI

## 📌 Project Overview

The **Superstore Sales Analysis** project analyzes retail sales data to generate meaningful business insights using **Python**, **SQL**, and **Power BI**. The project focuses on data cleaning, SQL-based analysis, and interactive dashboards that help understand sales performance, customer behavior, and product trends.

---

## 🎯 Problem Statement

Organizations generate large volumes of sales data every day, making it difficult to identify meaningful business insights. The objective of this project is to analyze Superstore sales data to understand sales performance, customer behavior, and product trends through data analytics and interactive dashboards.

---

## 💡 Solution

This project uses **Python**, **SQL**, and **Power BI** to transform raw sales data into meaningful business insights.

The workflow includes:

- Cleaning and preprocessing the raw dataset using Python (Pandas & NumPy).
- Exporting the cleaned dataset into CSV format.
- Importing the processed dataset into MySQL.
- Writing SQL queries to generate business insights.
- Building interactive dashboards in Power BI.
